配置 API Key & 模型
Key 与模型 ID 仅存储在本地浏览器，不会上传至任何服务器。填写后会自动保存；生成脚本、确认稿件等操作成功后也会再次写入，刷新页面后仍保留。具体模型由「模型 ID」决定。

推荐填入控制台 → 推理接入点 中创建的 ep-xxxxxxxx-xxxxx；也可填官方公开模型 ID（如 doubao-1-5-pro-32k-250115）。

在 platform.deepseek.com 获取

GPT ChatGPT（OpenAI）
OpenAI 官方 Key 在 platform.openai.com 获取；公司中转 Key（如 sk-mg-…）请同时填写下方「公司中转 API 地址」。
从公司网页复制 Base URL。只填域名即可（如 https://model.zhenguanyu.com），工具会自动补 /v1。
OpenAI 官方常用 gpt-4o、gpt-4o-mini；公司中转请按网页文档填写模型名。
OR OpenRouter
在 openrouter.ai/keys 获取；可用一个 Key 调多家模型，部分国外模型可能仍需出境网络。
模型 ID 格式为 vendor/model；完整列表见 openrouter.ai/models。
